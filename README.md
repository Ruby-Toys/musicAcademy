# Music Academy

## 개요

- 음악학원 스케줄 관리 프로젝트
- REST API 방식의 서버 구축 및 이해
- 개발 인원 : 1명
- 개발 기간 : 2022.08.22 ~ 2022.08.30

## Skill

- Java 11
- Spring boot
- Spring Security
- Spring Data JPA
- QueryDsl
- JUnit
- Spring Rest Docs
- MySql
- Vue.js

## 구현 기능

- 관리자
    - 관리자 로그인 / 로그아웃
- 수강생
    - CRUD
    - 결제 (카카오페이)
- 선생님
    - CRUD
- 스케줄
    - CRUD
    - 다음 날 스케줄에 따른 메일 전송
